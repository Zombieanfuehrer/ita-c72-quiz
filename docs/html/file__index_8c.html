◆ Index_Erweitern()

bool Index_Erweitern ( const char *  Dateiname)

Index_Erweitern fuegt in index.dat einen neuen Eintrag hinzu oder erstellt die index.dat sofern nicht vorhanden. In dieser Datei sind alle Dateinamen der gespeicherten "Fragen-Kataloge" hinterlegt.

Parameter
DateinameZeiger auf einen String mit dem anzuhaengenden Dateinamen des gespeicherten Fragen-Katalogs.
Rückgabe
true Index.dat wurde erfolgreich erweitert.
false Index.dat konnte nicht erweitert werden, details siehe stderr.
